## About the Project
SCHOLAR is a front-end web application designed to present educational content in an organized and visually appealing way. It allows users to explore courses, view instructor info, and navigate through different sections of the website seamlessly.

---

## Features
- **Course Listings** — Browse available courses with descriptions
- **Instructor Information** — Learn more about instructors and their expertise
- **Responsive Design** — Works on desktop, tablet, and mobile
- **About & Contact** — Contact form and information about the platform
